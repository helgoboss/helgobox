== Matrix sequencer

image:playtime::generated/screenshots/elements/toolbar/matrix-sequencer.png[Matrix sequencer, role="related thumb right"]

The matrix sequencer lets you record play and stop interactions with the matrix and write them to the arrangement when satisfied.
